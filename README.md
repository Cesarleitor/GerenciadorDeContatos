# Gerenciador de Contatos em Java

Este projeto é uma aplicação simples de console desenvolvida em **Java** para gerenciamento de contatos. O sistema permite cadastrar, listar, buscar e remover contatos diretamente pelo terminal, sendo uma ótima prática para reforçar conceitos fundamentais da linguagem.

## Funcionalidades

- Adicionar novos contatos
- Listar contatos cadastrados
- Buscar contato por nome
- Remover contato por nome
- Interação via terminal

## Tecnologias utilizadas

- Java
- Programação orientada a objetos
- Estruturas de dados com `ArrayList`
- Entrada de dados com `Scanner`

## Estrutura do projeto

O projeto está organizado com os seguintes arquivos na pasta `src`: `Contato.java`, `GerenciadorContatos.java` e `Main.java`. `Contato` representa a entidade com nome e telefone, `GerenciadorContatos` concentra a lista e as operações de adicionar, listar, buscar e remover, e `Main` executa o menu interativo no terminal. :contentReference[oaicite:1]{index=1}

## Conceitos praticados

- Classes e objetos
- Encapsulamento
- Métodos
- Listas com `ArrayList`
- Estruturas condicionais
- Estruturas de repetição
- Entrada e saída de dados no terminal
